## Block Button

By default the `.btn` is a inline-block button, we can add `.btn-block` to get the block button:

<input type="button" class="btn btn-block" value=".btn.btn-block"/>

```html
<input type="button" class="btn btn-block" value=".btn.btn-block"/>
```
